# Codex Change Guide

Use prompts like these when asking Codex to update the website later.

## Add Real Work Later

```text
Add a real completed work item for embroidered uniforms.

Use these images:
- image-one.webp
- image-two.webp

Category:
Embroidery

Services:
Custom Apparel

Only add it if the business has permission to publish the photos and details.
```

## Change the Hero Headline

```text
Change the homepage hero headline to:
Branded Apparel, Print and Signs for Growing GTA Businesses.

Keep the rest of the hero structure the same.
```

## Add a Service

```text
Add a new service page for promotional stickers.

Category:
Business Print

Include benefits, suitable applications, quote details needed, SEO title, meta description and approved image paths.
```

## Change Colours

```text
Update the accent orange in app/assets/css/tokens.css to a warmer red-orange.
Keep contrast accessible and do not change the full palette.
```

## Replace the Logo

```text
Replace the AC starter logo with the attached logo.
Update the header, footer, favicon and Open Graph image if needed.
```

## Add a GTA Location

```text
Add Burlington to the GTA service area list.
Update any map/list display if needed.
Do not create a separate location landing page.
```

## Update FAQs

```text
Add an FAQ explaining what files are best for embroidery.
Store it in content/faqs.json and show it on relevant apparel service pages.
```

## Change Phone Number

```text
Change the website phone number to the confirmed business number.
Update all references through content/site.json.
```

## Add a Social Link

```text
Add LinkedIn to the social links.
Use the confirmed company LinkedIn URL.
Show it anywhere existing social links are shown.
```
